Propiedades CSS

Propiedades generales

  • background-color

    • Sirve para colorear etiquetas del tipo caja

  • Padding

    • Establece el espacio de relleno requerido por todos los lados de un elemento.

  • Width

    • Controla la anchura de la caja de los elementos se denomina width . La propiedad width no admite valores negativos

  • Height

    • Especifica la altura del area de contenido de un elemento.

  • margin

    • Establece el margen para los cuatro lados.

Propiedades para texto

  • font-family

    • Define una lista de fuentes o familias de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado.

  • font-size

    • Se usa para modificar el tamaño de la fuente mostrada.

  • background-color

    • Permite mostrar un color de fondo sólido en la caja de un elemento.

  • text-aling

    • No sólo controla la alineación del texto. En realidad, esta propiedad establece la alineación de los contenidos (texto, imágenes) que se encuentran dentro de un elemento de bloque.

Propiedades para cajas

  • border

    • Permite definir de golpe todos los bordes en una única regla de la hoja de estilos.

  • Width

    • La caja no fuerza ningún salto de línea al llegar al final de la línea. Las propiedades width y height no se aplican. Se aplican relleno, margen y bordes verticales, pero no mantienen alejadas otras cajas en línea.

  • Height

    • A cada elemento de nivel bloque o elemento reemplazado se le puede dar una altura height, especificada como una longitud o como auto.
